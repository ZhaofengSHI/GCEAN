## Requirements

## PDVC

Ego2Exo

```
cd PDVC/
bash szf_scripts_new/ego2exo.sh
```

Exo2Ego

```
cd PDVC/
bash szf_scripts_new/exo2ego.sh
```

```
cd CM2/

```

Exo2Ego

```
cd CM2/
bash szf_scripts_new/exo2ego.sh
```
